Why choose MLR-X?

Portable and cross-platform

Self-contained binaries for Linux, macOS (Arm64), and Windows 10/11; no external Python or scientific stacks required.

GUI and CLI modes

GUI-only builds for quick exploration and GUI+CLI variants that keep the console open to run the full pipeline via config.conf.

Deterministic search engines

Original EPR-S heuristic and exhaustive all-subsets search with multicollinearity gates, reproducible runs, and configurable seeds.

Validation and diagnostics

Built-in LOO and k-fold validation, external validation, diagnostics, visualization, and summary exports synchronized across tabs.

Open source

Distributed under AGPL-3.0; contribute, review the code, and build extensions on an open base with citation guidance.
