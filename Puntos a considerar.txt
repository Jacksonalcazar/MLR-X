Puntos a considerar

1) Identidad y alcance

Nombre y versión

Licencia y derechos: AGPL-3.0, con derechos 2025; información incluida en los metadatos de la distribución Windows.

Público objetivo: usuarios que requieran ajustar y diagnosticar regresiones lineales múltiples en datasets pequeńos o grandes.

2) Canales oficiales y recursos

Página y repositorio: enlace al repositorio GitHub y página del proyecto.

Manual oficial en PDF (versión 1.0).

Soporte y reporte de incidencias vía GitHub Issues.

Donaciones: enlace PayPal oficial.

3) Distribuciones y compatibilidad

Builds disponibles: Linux (binario), Windows GUI y Windows GUI+CLI; build macOS en preparación.

Versión del producto en ejecutables: 1.0, con file/product version 1.0.0.0 para Windows.

Dependencias pesadas cargadas en segundo plano (numpy, pandas, statsmodels, matplotlib, scikit-learn) para rendimiento y portabilidad.

4) Modos de uso

Interfaz gráfica (GUI) para exploración rápida y diagnósticos visuales.

Línea de comandos (CLI) para pipelines reproducibles.

Paquete combinado Windows incluye GUI + CLI.

5) Funcionalidad principal

Ajuste y diagnóstico de regresión lineal múltiple con soporte para métricas estándar (MAE, MSE, R˛) y exportación de resultados.

Análisis escalable: ejecución paralela y carga diferida de librerías pesadas.

Umbrales y opciones clave configurables (ej.: covarianza robusta, número de trabajos en paralelo).

Texto de citación recomendado y entrada BibTeX para versión 1.0.

6) Documentación que debe incluir el manual

Requisitos de sistema por plataforma (basado en builds listadas y dependencias).

Guía de instalación para cada paquete oficial (Linux binario, Windows GUI, Windows GUI+CLI; nota sobre disponibilidad futura de macOS).

Guía rápida de inicio: ejemplos de CLI y recorrido de la GUI.

Configuración avanzada: opciones de paralelismo (n_jobs), selección de métricas objetivo, modos de iteración y covarianza.

Exportación y reportes: formatos esperados y rutas de salida.

Solución de problemas y preguntas frecuentes: referencia a Issues oficiales y verificación de dependencias pesadas.

Créditos y citación académica (texto y BibTeX), junto con aviso de licencia AGPL-3.0.

Historial de versión y cambios destacados de la 1.0 (incluir release inicial, canales de descarga, soporte GUI/CLI)
